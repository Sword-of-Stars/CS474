\section{DFA Transition Function}

The DFA's transition function, $\delta$, is defined as follows:
\begin{align*}
{%- for state, transitions in tf.items() -%}
    {%- for symbol, target in transitions.items() -%}
        \delta({{ state }},\,{{ symbol }}) &= {{ target }} {%- if not loop.last %}\\[0.5em]{%- endif -%}
    {%- endfor -%}
{%- if not loop.last %}\\[1em]{%- endif -%}
{%- endfor -%}
\end{align*}

\subsection*{Summary}

The table-filling algorithm successfully identified all distinguishable state pairs through systematic analysis:
\begin{itemize}
\item \textbf{Initialization (Step 1):} Marked pairs with different finality status through cumulative analysis as shown in Figure~\ref{fig:indis_table_step1}
\item \textbf{Refinement steps:} Iteratively marked pairs whose transitions lead to already-distinguishable pairs
\item \textbf{Convergence:} Reached fixed point when no new pairs could be marked
\item \textbf{Result:} Minimized DFA with merged equivalent states (Figure~\ref{fig:minimized_dfa})
\end{itemize}
